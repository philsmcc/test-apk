This page demonstrates how to customize the chat widget appearance with your own CSS.

Available Themes

Blue Theme
Green Theme
Purple Theme
Dark Theme

This example uses the green theme with a custom pulsing animation on the chat button.

How to Customize

Add custom CSS after including the widget script to override the default styles:

<style>
.openwebui-chat-button {
    background: #your-color !important;
    animation: pulse 2s infinite;
}

.openwebui-chat-header {
    background: #your-color !important;
}
</style>

You can also modify the CONFIG object in chat-widget.js to change the default theme, position, messages, and more!
