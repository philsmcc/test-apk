Demo Instructions

This page demonstrates the Open WebUI chat widget. Look for the blue chat button in the bottom-right corner of the page.

  • Click the chat button to open the popup
  • Type a message and press Enter or click the send button
  • The widget connects to your Open WebUI instance
  • Uses your customized smollm2:360m model
  • Maintains conversation history during the session

How to Integrate

To add this chat widget to your website, simply include the JavaScript file:

<script src="chat-widget.js"></script>

That's it! The widget will automatically initialize when your page loads.

Configuration

You can customize the widget by modifying the CONFIG object at the top of the chat-widget.js file:

const CONFIG = { // Your Open WebUI instance URL openWebUIUrl: 'http://108.178.153.147:80', // Model to use for chat model: 'smollm2:360m', // Widget appearance position: 'bottom-right', // bottom-right, bottom-left, top-right, top-left theme: 'blue', // blue, green, purple, dark // Chat settings placeholder: 'Ask me anything...', welcomeMessage: 'Hello! How can I help you today?', };

Features

  • Responsive design that works on desktop and mobile
  • Clean, modern UI with smooth animations
  • Automatic authentication handling
  • Session management and conversation history
  • Customizable themes and positioning
  • Error handling and loading states
  • Easy integration - just one script tag
Note: Make sure your Open WebUI instance is running and accessible. The current configuration points to http://108.178.153.147:80 and uses the smollm2:360m model.

Customization Options

Themes Available:

  • blue - Default blue theme
  • green - Green theme
  • purple - Purple theme
  • dark - Dark theme

Position Options:

  • bottom-right - Bottom right corner (default)
  • bottom-left - Bottom left corner
  • top-right - Top right corner
  • top-left - Top left corner

Advanced Usage

You can also programmatically interact with the widget:
